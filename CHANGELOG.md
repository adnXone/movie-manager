# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-19

### Added
- Initial release of movie-manager skill
- Watchlist and seen list management
- Personalized movie recommendations
- YouTube trailer integration
- Post-viewing reflection templates
- +1/-1 feedback system
- Romanian language support (LANG=ro)
- English language support (LANG=en)
- OpenClaw and ClawHub compatibility
- Obsidian WikiLink integration
- Movie metadata tracking (ratings, streaming platforms)

### Configuration
- Configurable language via `LANG` variable
- Customizable directory structure
- Support for movie poster URLs
- Cron integration for reminders
