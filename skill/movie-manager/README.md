# Movie Manager pentru OpenClaw

Organizează recomandările de filme în Obsidian și oferă sugestii personalizate.

## Funcționalități cheie
- Recomandări bazate pe profilul tău (actori, gen, teme)
- Trailere YouTube obligatorii
- Liste de urmărit (watchlist) și văzute (seen)
- Notație wiki-links pentru observanțe și învățare activă
- Fișiere cu nume: `Titlu (An) - Actor1, Actor2.md`

## Instalare
```bash
clawhub install movie-manager
```

## Structură fișier
```markdown
# Titlu (An)
- Descriere
- Actori: [[Actor1]], [[Actor2]]
- Gen: [[Gen]]
- Trailer: [YouTube](...)

### De ce ți s-ar potrivi
Text personalizat
```
